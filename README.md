# Fresh Island Foods Dashboard — Leeds, LS10 1BL

Upload all files in this folder to the root of the matching GitHub repository.

The visual design is unchanged. This version adds:
- animated weather icons
- live weather refresh every 10 minutes
- a soft light sweep every roughly 8–14 minutes
- a two-pixel whole-screen shift every five minutes

The movement is subtle and requires no interaction. It helps reduce static-image
retention, although no webpage can guarantee prevention of permanent burn-in.
